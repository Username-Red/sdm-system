## Purpose 

Many Schools still manage students’ data manually thereby leading data loss, data duplication, difficulty in accessing information, time delay and even errors. This Student Data Management System aims to ameliorate these problems by providing a web-based platform to effectively and efficiently manage students’ records. 

The management system will allow school administrators to easily add, view, update, and delete student record while maintaining data accuracy, security, and easy access from any internet-connected device. 

 

## Features 

List the main features of your project 

User Authentication: A login system for school administrators to access and manage records 

Add Student Record: Get students’ information such as name, age, gender, class, address, phone number, email etc 

View Student List: Show a list of registered students 

Edit Student Record: Easily update an existing student record 

Delete Student Record: Easily remove student record who has left the school 

Search Bar: To quickly find student record either by name, class or id 

Responsive UI: Works responsively on large and small screen 

Data Storage: Stored and accessible online
